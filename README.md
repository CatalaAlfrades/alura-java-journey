Welcome to the **[alura-java-journey](https://github.com/CatalaAlfrades/alura-java-journey)** repository! ☕

This repository was created to document my entire learning journey in the **Java Back-End Developer** career path at [Alura](https://www.alura.com.br/). Here you will find the source code, notes, exercises, and practical projects developed throughout the courses.

## 🎯 Objective

The main goal of this repository is to consolidate the knowledge acquired during the training, serving as a practical portfolio of my evolution in the Java language, the Spring Boot ecosystem, and software engineering best practices.

## 📚 Learning Structure (Java Back-End Track)

The projects contained here reflect the main pillars of the Alura Java training:

### 1. Java Fundamentals and Object-Oriented Programming
- First steps with Java: Variables, Data Types, and Control Flow.
- **Object-Oriented Programming (OOP):** Classes, Objects, Encapsulation, Inheritance, Polymorphism, and Abstraction.
- Exception Handling.
- String Manipulation, `java.io`, and base packages.
- Java Collections (Lists, Sets, Maps) and Generics.

### 2. Web Development with Spring Boot
- Creation of structured RESTful APIs.
- Dependency Injection and Inversion of Control.
- **Data Persistence:** Spring Data JPA, Hibernate, and database versioning with Flyway.
- **Security:** Spring Security, Authentication, and Authorization using JWT (JSON Web Tokens).
- API best practices: Pagination, Sorting, Validations (Bean Validation), and Global Error Handling (Controller Advice).

### 3. Architecture, Design, and Best Practices
- Understanding and application of **SOLID** principles.
- Design Patterns applied to the real world.
- Clean Architecture concepts.
- Legacy code refactoring and standardization.

### 4. Automated Tests
- Creation of Unit Tests and Integration Tests.
- Using **JUnit 5** and **Mockito** for structured mocks.

## 🛠️ Technologies and Tools Used

- **Language:** Java (version 17+)
- **Main Framework:** Spring Boot 3
- **Dependency Manager:** Maven
- **Database:** MySQL / PostgreSQL
- **Migrations:** Flyway
- **Testing:** JUnit 5, Mockito
- **IDE:** IntelliJ IDEA / Eclipse
- **Version Control:** Git & GitHub
